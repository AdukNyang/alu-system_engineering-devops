# Web Server Project

This project contains scripts to configure an Nginx web server.

## Tasks

0. Transfer a file to your server - SCP file transfer script
1. Install nginx web server - Install and configure Nginx
2. Setup a domain name - Domain configuration
3. Redirection - Configure 301 redirect
4. Not found page 404 - Custom 404 error page
5. Design a beautiful 404 page - Creative 404 page
